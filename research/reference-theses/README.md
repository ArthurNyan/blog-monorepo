# Reference Theses Archive

Архив похожих дипломных и исследовательских работ для подготовки ВКР по теме CMS-first маркетинговой платформы на базе `Strapi` и `Astro`.

Дата выгрузки: `2026-05-17`

## Структура

- `downloads/` — локальные копии PDF, если источник открыт.
- `metadata/` — сохраненные карточки источников, если полная работа недоступна напрямую или полезно сохранить страницу-описание.
- `notes/` — вспомогательные заметки для дальнейшей работы.
- `catalog.md` — сводная таблица по источникам и полезности.

## Принцип отбора

В архив вошли:

- работы про `headless CMS`, `API-driven CMS`, мультиязычные веб-приложения и контентные платформы;
- близкие по теме бакалаврские работы и тезисы из ИТМО, ТПУ, СПбПУ;
- локальные кафедральные референсы РГПУ, добавленные вручную из пользовательских PDF;
- сильные зарубежные open-access работы, которые дают более современный уровень архитектурной и проектной проработки.

## Ограничения

- Не все работы доступны как полный PDF. Для таких источников сохранены карточки с метаданными.
- Часть российских источников полезна скорее как пример постановки задачи и структуры, а не как современный технологический эталон.
- Часть зарубежных работ ближе к `React`, `Qwik`, `GraphQL` или собственным CMS-реализациям, а не к `Astro`, но остаются релевантными по архитектуре, публикации и data model.

## Быстрый старт

1. Открыть `catalog.md`.
2. Начать с работ с релевантностью `5/5`.
3. Затем перейти к `thesis/knowledge/diploma/reference-theses-analysis.md` для выводов, адаптированных под текущий диплом.
