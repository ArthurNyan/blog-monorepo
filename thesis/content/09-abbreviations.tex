\begin{description}
  \item[API] Application Programming Interface — интерфейс программирования приложений.
  \item[CMS] Content Management System — система управления контентом.
  \item[CI/CD] Continuous Integration / Continuous Delivery — непрерывная интеграция и доставка.
  \item[CRUD] Create, Read, Update, Delete — базовые операции управления данными.
  \item[DTO] Data Transfer Object — объект передачи данных между слоями приложения.
\end{description}
