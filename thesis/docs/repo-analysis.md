# Анализ референсных репозиториев

Дата среза: 14.03.2026.

## Выводы по репозиториям

### 1) LaTeX-GOST-BMSTU-Diploma
- Плюсы: понятные примеры титульных листов, базовые ГОСТ-поля, практическая ориентация.
- Минусы: проект ориентирован на XeLaTeX, встречаются устаревшие/хрупкие настройки
  для современного Unicode-движка (включая параметры листингов и точечные правки отступов).

### 2) itonik/spbu_diploma
- Плюсы: аккуратная структура и удобная идея \texttt{specialsection}.
- Минусы: legacy-стек (\texttt{inputenc/fontenc/babel}), ориентация на локальные требования СПбГУ,
  устаревшие решения для современных LuaLaTeX-пайплайнов.

### 3) maks2199/DiplomaTeX
- Плюсы: модульная структура, развитая преамбула, применение \texttt{biblatex-gost}.
- Минусы: переусложненная преамбула, дублирование пакетов, XeLaTeX-first конфигурация,
  избыточные зависимости для MVP-шаблона.

## Что заимствовано
- Модульная структура проекта (main + sections/content).
- Практика вынесения пользовательских данных в отдельный конфиг (в этом проекте через \texttt{\gostsetup}).
- Использование \texttt{biblatex-gost} + \texttt{biber}.

## Что сознательно отброшено
- Legacy-пакеты и кодировки для Unicode-движков (\texttt{inputenc/fontenc} в LuaLaTeX-профиле).
- Жесткая привязка к локальному вузовскому титулу.
- Переусложненные и дублирующиеся настройки преамбулы.

## Итоговое решение
Собран компактный LuaLaTeX-first шаблон с единым классом \texttt{gostvkr-lua.cls},
ориентированный на Core-4 baseline и быструю воспроизводимую сборку через \texttt{latexmk}.
