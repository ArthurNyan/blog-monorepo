# Аудит шаблона преподавателя

## Источник

- Репозиторий: `remote-worker-prof/edu-thesis-gost`
- Локальная копия: корень папки `thesis/`

## Базовые факты

- Шаблон построен на классе `gostvkr-lua`.
- Основной движок строго зафиксирован: `LuaLaTeX`.
- Библиография собирается через `biber`.
- Основной entrypoint: `main.tex`.
- Основные данные титула задаются через `config/metadata.tex`.
- Структура документа зашита вокруг двух нумеруемых глав, введения, заключения и приложений.

## Сборка

`Makefile` шаблона ожидает следующие инструменты:

- `latexmk`
- `lualatex`
- `biber`
- `chktex`
- `lacheck`
- `pdftotext`
- `pdffonts`

Ключевые цели `make`:

- `make build` — сборка `main.pdf`
- `make rebuild` — полная пересборка
- `make watch` — непрерывная сборка
- `make lint` — `chktex + lacheck`
- `make check` — проверка PDF-маркеров, лога и ассетов
- `make fontcheck` — проверка наличия и отказоустойчивости шрифтов
- `make ci` — полный цикл проверки

## Шрифты и ассеты

- Основной текстовый шрифт: `PT Astra Serif` из локальной папки `fonts/`.
- Шрифт листингов: `Fira Code` из `fonts/fira-code/`.
- Титульный лист использует локальный логотип `assets/title-2025-logo.png`.
- Класс аварийно останавливает сборку, если обязательные шрифты не найдены.

## Что важно для текущего проекта

- Шаблон импортирован как snapshot и не связан с `apps/front` или `apps/cms`.
- `content/*.tex` и `Makefile` оставлены без содержательных изменений для baseline-сборки.
- `config/metadata.tex` обновлен только безопасными стартовыми данными.
- Для текущей машины `Makefile` дополнительно подхватывает локальный `TinyTeX` из `~/Library/TinyTeX/bin/universal-darwin` и `poppler` из `/opt/homebrew/bin`, чтобы `make build` работал без ручного экспорта `PATH`.

## Риски и ограничения

- Без установленного TeX toolchain `make build` не запустится.
- `make check` содержит контентно-жесткие проверки по sample markers, поэтому после сильной переработки содержания его может понадобиться адаптировать.
- Шаблон не обещает полное совпадение с формальными требованиями РГПУ по шрифту и полям.
